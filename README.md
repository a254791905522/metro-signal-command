# Metro Signal Command Release Package

App name: Metro Signal Command

Bundle ID: com.rosewood.driftsubway.game

Device support: iPhone only

Gameplay fact card:
- Core action: accelerate and brake metro trains through 60-second midnight shifts.
- Main puzzle: manage 1-4 trains per line with speed-limit zones and following-distance collision avoidance.
- Fault response: resolve signal, door, and crowd faults before the timer ends.
- Progression: 50 levels across 50 world cities (1372 real stations), scaling from 1 to 4 trains.
- Monetization and network: offline, no ads, no in-app purchases, no account, no analytics, no tracking.
